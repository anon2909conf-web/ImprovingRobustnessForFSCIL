Multi-Mode Feature Augmentation

Robust Feature Distributions for Few-Shot Incremental Learning

🔥 Abstract

We propose a novel feature augmentation strategy for Few-Shot Class-Incremental Learning (FSCIL). Our method generates multi-mode feature distributions using spatial mixing, frequency-domain perturbations, and controlled noise to improve class separability while preserving decision boundaries.

✨ Key Contributions

🎯 Multi-Mode Augmentation

Spatial + frequency + noise-based feature enrichment.

Improves intra-class diversity without boundary collapse.

⚙️ How It Works

  1. Extract base features using backbone network
  2. Apply spatial feature mixing
  3. Apply frequency-domain perturbation
  4. Add controlled Gaussian noise
  5. Normalize and train with contrastive loss

🚀 Conclusion

Multi-mode augmentation significantly improves feature robustness in FSCIL, enabling better generalization to unseen classes while preserving old knowledge.
